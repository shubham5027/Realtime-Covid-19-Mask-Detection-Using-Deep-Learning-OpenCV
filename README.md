        <h1>Face Mask Detection System</h1>
        <p>Welcome to the repository of our cutting-edge Face Mask Detection System that uses OpenCV and CNN.</p>

        <h2>Project Overview</h2>
        <p>This project aims to offer a reliable solution to determine if individuals are wearing face masks correctly in real-time.</p>

        <h3>Features</h3>
        <ul>
            <li>High Accuracy: Near 97% accuracy with our custom CNN.</li>
            <li>Real-Time Detection: Live video stream processing using OpenCV.</li>
            <li>Two-Class System: Identifies 'with mask' and 'without mask' cases.</li>
            <li>User-Friendly Interface: Minimal setup required for running the project.</li>
        </ul>

        <h2>How It Works</h2>
        <p>The system uses a CNN to categorize faces as 'with mask' or 'without mask', and then applies this model in real-time using OpenCV.</p>
